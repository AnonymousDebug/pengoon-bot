Welcome to Ball Discord Bot's documentation

Adding our Public Bot

Don't want to host it yourself? We got you covered! You can add our public bot and relax.

Step-by-step instruction

  1. Click here.
  2. Select the guild you want to add the bot to and click "Continute" on the bottom right of the panel.
  3. Go to the bottom right of the panel and click "Authorize".
  4. You may need to do a captcha, then the bot should be added to your guild.

Self-Hosting

Environment Variables

  • TOKEN: Discord Bot Token.
  • RANDOMMER_API: The API key for randommer.
  • POSTGRES_HOST: The hostname for your PostgreSQL database.
  • POSTGRES_DATABASE: The name of your PostgreSQL database.
  • POSTGRES_USER: The username for your PostgreSQL database.
  • POSTGRES_PASSWORD: The password for your PostgreSQL database user.

Hosting locally

  1. Clone the repository.
  2. Install any missing dependencies.
  3. Get into the poetry virtual environment.
  4. Start the bot.
$ git clone https://github.com/MouseMoosz/ball.git
$ poetry install
$ poetry shell
$ python3 src/bot.py
