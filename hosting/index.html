Prerequisites

Creating a discord bot account

  1. Create a new application.
  2. Give it a nice name and create.
  3. Click on the bot section.
  4. Add a bot to your application.
  5. Confirm.
  6. Get the bot token.
  7. Confirm.
  8. Copy the token (The token shown in the image is not valid anymore).
  9. Scroll down and enable Server Members Intent and save changes.
  10. Click on the OAuth2 tab.
  11. Click on Url Generator tab inside the OAuth2 tab.
  12. Select "bot" and "application.commands" scopes and "Administrator" permission (for the sake of getting started).
  13. Scroll down and copy your generated URL.
  14. Visit that URL and add the bot to your guild like shown here.
  15. Add the bot token as a Environment Variable.

Environment Variables

  • TOKEN: Discord Bot Token
  • RANDOMMER_API: The API key for randommer
  • POSTGRESQL_HOST: The hostname for your PostgreSQL database
  • POSTGRESQL_DATABASE: The name of your PostgreSQL database
  • POSTGRESQl_USER: The username for your PostgreSQL database
  • POSTGRESQL_PASSWORD: The password for your PostgreSQL database user
